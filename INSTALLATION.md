# INSTALLATION INSTRUCTIONS
If you are looking to install this project, there are three methods, with method 1 being the preferred way to install.
## METHOD 1 (RECOMMENDED)
To install pytp, you can go to the releases page that is found on the main page. There download the attached Python file and run it. It's as simple as that. If you haven't installed Python, you can download it here: [python.org/downloads](https://www.python.org/downloads/)
## METHOD 2
If you don't want to install Python, you can download the executable file instead. Please note that pyinstaller executable files are generally slower than normal Python scripts because they bundle the interpreter along with the program. Use this if you cannot install Python.
## METHOD 3
**⚠️ NOTE:** This method is not recommended.  
If you for some reason really don't want to download a script file or executable, you can download the whole project for yourself. Just click the `Code` button and click `Download ZIP`. However, you won't just download the program, you'll download the whole project, including the GitHub readme.
