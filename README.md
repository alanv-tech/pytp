# Python Time Protocol
A simple python network time protocol server with simulated clients to test functionality.
### 🌐 WEBSITE
If you got here from the GitHub pages website, just click the blue title at the top to go to the repository page. The pages website can be outdated at certain times, so it is recommended to go to the repository webpage.
## FEATURES
<details>
  <summary><b>🕑 ADVANCED FUNCTIONALITY</b></summary>
  
  This Python program simulates a network time protocol server right from your computer using localhost port 8080. It comes with an advanced log that can keep track of all connected clients and activity.
</details>
<details>
  <summary><b>😊 EASE OF USE</b></summary>
  
  It comes with a well polished TkInter GUI with easy to read fonts, and all program functionality is self explanatory.
</details>
<details>
  <summary><b>💻 SIMULATED CLIENTS</b></summary>
  
  Don't have any users that want to connect to your server? Well worry no more, as this program comes with complete simulated clients that resemble real users from around the world connecting to your server. The program allows you to choose between different time zones depending on where you live, and it offers an auto-refresh system that can automatically fetch the time without you having to press a button.
</details>
<details>
  <summary><b>🔀 CLIENT SYNCHRONIZATION</b></summary>
  
  Noticing that your client time offset is becoming too big? This program comes with a feature that can synchronize the times of all connected clients. 
</details>
